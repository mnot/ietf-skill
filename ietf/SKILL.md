# Participating in IETF and IRTF work

This skill collects the norms for working within the IETF and IRTF. It is
organised by capability: find the task in front of you below and read that
capability's file before you act. Each file is self-contained.

## Capabilities

- **Contributing** — drafting text that will go into the record under a
  participant's name: mailing-list mail, replies in a thread, GitHub issues or
  comments, reviews, or other discussion contributions. Read
  [references/contributing.md](references/contributing.md) BEFORE drafting any
  such text. Authoring Internet-Drafts is out of scope.

_Further capabilities will be added here as separate files; load only the one
that matches the task._
